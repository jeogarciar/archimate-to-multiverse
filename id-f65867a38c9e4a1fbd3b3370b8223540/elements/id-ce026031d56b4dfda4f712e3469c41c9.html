POST /api/auth ()

API REST
API expuesta mediante un método HTTP POST la cual tiene como objetivo la generación de un token de seguridad. { "Username":"usuario", "Password":"password" } Dicho JSON deberá cifrarse bajo el algoritmo AES 256 y el valor obtenido deberá ser enviado en una propiedad Content en el body de la petición { "Content":"VH6y3rU+6rYiWjHwGtRrNcUJ8fXxTGIm3uwOqRjdYWMXaYvh2j+YDVbc1n0G0+hp04e6tLve 5eBe4Nu1tBKz2Q==" } La respuesta obtenida deberá des-cifrarse igualmente bajo el algoritmo AES 236. { "content":"VH6y3rU+6rYiWjHwGtRrNcUJ8fXxTGIm3uwOqRjdYWMXaYvh2j+YDVbc1n0G0+hp04e6tLve 5eBe4Nu1tBKz2Q==" } El valor obtenido después de ser des-cifrada contiene tres propiedades IsSuccess, UserCryptoKey y ResultData. IsSuccess representa mediante un valor booleano el resultado de la transacción. UserCryptoKey representa una llave en formato texto que deberá ser usada para cifrar o des-cifrar otras peticiones. ResultData representa el token JWT que debe ser usado.
