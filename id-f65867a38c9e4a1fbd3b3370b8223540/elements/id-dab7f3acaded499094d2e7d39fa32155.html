Página gamer ()

Objetivo
Se desea que se implemente una página en ETB.COM que pueda ser accedida mediante un usuario y contraseña de MiETB y proyecte la información de soporte gamer, las últimas tendencias, y enlaces adicionales. Sin embargo, a dicha solicitud realmente se le va realizar una varianza. Las páginas serán implementadas en el gestor de contenido pero serán accedidas desde una petición HTTP GET en MiETB.
