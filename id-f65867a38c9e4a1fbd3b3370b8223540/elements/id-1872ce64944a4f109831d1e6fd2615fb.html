PUT /api/user ()

API REST
API expuesta mediante un método HTTP PUT la cual tiene como objetivo la actualización de un usuario creado previamente. El contenido de la petición debe incluir los siguientes campos: { "Name":"nombre", "Surname":"apellido", "Surname2": "surname2", "Email": "Email", "Eid": "Licencia", "Cif": "cif", "Address": "ad", "Phone":"phone" } El valor de este JSON debe ser cifrado en AES 256 antes de ser enviado a la petición. Adicionalmente se requiere de la autenticación JWT. La respuesta de la petición deberá ser des-cifrada igualmente bajo el algoritmo AES 256. El resultado retorna los campos que fueron actualizados para el usuario.
