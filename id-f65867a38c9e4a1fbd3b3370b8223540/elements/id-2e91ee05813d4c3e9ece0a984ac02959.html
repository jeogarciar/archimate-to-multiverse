POST /api/v1/fractalia/ ()

API REST
API expuesta mediante un método POST que permite crear un uusario en Fractalia. El contenido de la petición debe incluir la información básica de una persona, la licencia que será adquirida y los datos de contacto. Por defecto los campos obligatorios son : nombre, apellido, licencia, identificacion, correo electrónico, telefono y dirección. { "nombre":"nombre", "apellido":"apellido", "segundoApellido": "segundo apellido", "licencia": "licencia", "identificación": "cif", "contacto": { "correoElectronico": "correo electrónico", "direccion": "dirección de residencia", "telefono":"telefono de contacto" } } La respuesta de la petición debe incluir únicamente en la cabecera el código del estado http estándar: 200 para OK, 500 para algún error ejecutado en el servidor, entre otras.
